# UPSC General Studies Practice Quiz

This 50-question quiz is modeled on recurring UPSC GS trends from the last decade.
Each question includes a brief reference line showing:

- `PYQ trend`: the kind of theme that has repeatedly appeared in UPSC papers
- `Core source`: the standard-book area the question is built from
- `Current hook`: the recent-affairs angle used to make it UPSC-relevant
